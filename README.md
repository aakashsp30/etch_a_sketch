# Etch-a-Sketch

A browser-based drawing game where you sketch by hovering your mouse over a grid of squares.

## Features

- **Colorful Drawing**: Each square turns a random color when you hover over it
- **Progressive Darkening**: Hover over the same square multiple times to make it darker (10% per hover, black after 10 hovers)
- **Custom Grid Size**: Click "New Grid" to create grids from 1x1 up to 100x100

## How to Use

1. Move your mouse over the grid to draw
2. Click "New Grid" to change the grid size
3. Enter a number between 1-100
4. Start drawing on your new grid!
